# 🎮 Rock Paper Scissors Web

A modern and interactive Rock Paper Scissors game built using **HTML, CSS, and JavaScript**. Challenge the AI, watch animated hand battles, and enjoy a clean gaming experience directly in your browser.

## ✨ Features

* 🎨 Responsive and modern UI
* ✊ Animated hand-shake effect
* 🤖 AI opponent
* 📊 Live score tracking
* 🏆 Winner announcement
* ⚡ Instant gameplay
* 🎯 Rock, Paper, Scissors controls
* 💬 AI taunts and game messages
* 🌐 Runs directly in the browser

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6)

## 🎮 How to Play

1. Open `rock-paper-scissors.html` in your browser.
2. Click:

   * Rock ✊
   * Paper ✋
   * Scissors ✌️
3. Watch the hand animation.
4. See the AI's move.
5. Track the score and try to win!
